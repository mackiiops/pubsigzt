How to add it

One copy, then paste it wherever your signature lives.

1

Gmail

Settings (gear) → See all settings → General → Signature → Create new, then paste with Copy signature.

2

Outlook (desktop)

File → Options → Mail → Signatures → New, then paste with Copy signature.

3

Outlook web / HubSpot

Use Copy HTML code anywhere an HTML signature is accepted.
